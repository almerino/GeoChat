Geochat
====================
This Android app is a school project.

Technologies used
---------------------
- Java
- Android sdk
- Google Map API

Features
---------------------
- Wifi connection (Host/Client)
- Channel creation, public and private mode
- Channel chat and multiple user private chat 
- File sharing
- Geopositionning
- Wizz
